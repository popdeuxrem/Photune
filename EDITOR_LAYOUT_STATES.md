# EDITOR_LAYOUT_STATES

## Required Editor States

### 1. New Project / No Image
- empty canvas state
- prominent upload CTA
- inactive AI tools explained

### 2. Image Loaded / No Selection
- canvas dominant
- editing tools available
- contextual instructions

### 3. Text Selected
- text actions become primary
- rewrite/translate/font tools visible contextually

### 4. Object Selected
- object/layer actions visible contextually

### 5. Export State
- compact export modal/sheet
- format choices
- success/failure feedback

### 6. Mobile Tool Panel Open
- canvas still visible
- panel does not permanently consume majority width
- easy dismiss gesture
