# ADD_TEXT_ACTION_PLAN

## Goal
Complete the basic text-editing loop in Photune.

## Scope
- add a reusable text creation helper
- expose an "Add Text" action in Text mode
- create/select a Fabric text object
- persist via saveState()
- switch/focus into Text mode controls

## Success criteria
- user can create a new text object from Text mode
- new object is tagged with semantic layer metadata
- new object is selected immediately
- text controls become active
- validation stays green
