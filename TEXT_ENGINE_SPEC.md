# TEXT_ENGINE_SPEC

## Goal
Define the next-stage typographic studio without destabilizing current text editing.

## Near-term scope
- alignment controls
- font size
- tracking
- line height
- color
- text selection awareness

## Deferred scope
- variable font axes
- OpenType feature toggles
- OCR-based spacing inference
- font preload orchestration beyond current needs
- color sampling from source image

## Rule
Text controls should remain contextual and selection-aware.
Do not block the whole canvas on font loading.
